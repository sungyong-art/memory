# 📋 작업 브리프

**원 명령:** 간단한 포모도 시계 앱 하나 만들어

## 요약
사용자의 요청에 따라 간단한 포모도로 타이머 애플리케이션을 개발합니다. 핵심 기능은 시간 카운트다운과 상태 전환 로직 구현입니다.

## 분배
- **💻 Developer**: 포모도로 타이머 앱의 초기 코드를 작성하세요. 웹 기반(HTML, CSS, JavaScript)으로 구성하며, 다음 기능을 포함해야 합니다: 1. 카운트다운 기능 (25분 기본 설정). 2. 세션 상태 전환 버튼 (작업 시간 → 휴식 시간 → 완료). 3. 남은 시간을 시각적으로 표시하는 인터페이스 로직.
- **✍️ Writer**: 앱의 사용자 인터페이스(UI)에 들어갈 명확하고 간결한 문구를 작성하세요. 예: 앱 제목, '시작' 버튼 안내 문구, 세션별 상태 메시지('집중 모드!', '잠시 휴식하세요').
